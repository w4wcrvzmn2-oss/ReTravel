# ✈ ReTravel

Агрегатор туров, авиабилетов и отелей. Сравниваем предложения от туроператоров — вы выбираете лучшее.

## Обзор проекта

https://github.com/w4wcrvzmn2-oss/ReTravel/raw/main/isk/export-1782228958812.mp4

## Стек

- **React 18** + **Vite 5** — SPA с быстрой сборкой
- **Tailwind CSS 3** — утилитарные стили, тёмная и светлая тема
- **Zustand 4** — стейт-менеджмент с persist-middleware (localStorage)
- **React Router DOM v6** — клиентская маршрутизация
- **Lucide React** — иконки

## Функциональность

| Раздел | Описание |
|---|---|
| 🏠 Главная | Поиск-бар с вкладками, популярные направления, горящие туры |
| 🔍 Поиск туров | Фильтры (категория, питание, звёзды, цена), сортировка, пагинация |
| 🌍 Направления | Каталог направлений, страница каждого направления с турами |
| ✈ Авиабилеты | Поиск рейсов с фильтром по направлению и классу |
| 🏨 Отели | Каталог с фильтром по звёздам и направлению |
| 🗺 Тур | Галерея с лайтбоксом, отзывы, подписка на снижение цены |
| ⚖ Сравнение | Таблица сравнения до 3 туров |
| 🛒 Корзина | Промокоды (RETRAVEL5, WELCOME10, HOT20), оформление заказа |
| ❤ Избранное | Сохранение туров, вкладка в личном кабинете |
| 👤 Кабинет | История заказов, избранное, профиль |
| ℹ О нас | Ценности компании |
